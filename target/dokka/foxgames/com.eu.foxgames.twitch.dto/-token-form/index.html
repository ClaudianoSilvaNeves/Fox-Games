TokenForm

data class TokenForm(access_token: String, expires_in: Long, token_type: String)

Data transfer object (DTO) representing the token information received from Twitch.

This class is used to transfer the token information (such as access token, expiration time, and token type) from the API responses to the application logic. It provides getter methods to access the token details.

Parameters

access_token

The access token received from the Twitch API.

expires_in

The expiration time of the token in seconds.

token_type

The type of the token (e.g., Bearer).
