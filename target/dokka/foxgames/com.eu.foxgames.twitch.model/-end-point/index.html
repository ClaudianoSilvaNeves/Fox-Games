enum EndPoint : Enum<EndPoint>

Enum class representing different API endpoints for querying game-related data from Twitch.

Each value in this enum corresponds to a specific endpoint in the Twitch API that deals with different aspects of game data, such as game ratings, covers, platforms, genres, and much more. The url() function converts the enum constant into its lowercase string representation, which can be used to build a URL for making API requests.
