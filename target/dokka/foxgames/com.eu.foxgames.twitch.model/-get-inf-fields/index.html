Data class that holds the fields for a game query.

This class is used to define a list of fields that can be requested from the Twitch API. The list contains various attributes related to a game, such as ratings, platforms, release dates, and more. The gameFields property contains a comma-separated string of these fields, which can be used in queries to specify which data should be returned by the API.
